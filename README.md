Aplikasi CRUD Laravel 12 untuk manajemen inventaris barang elektronik laboratorium.

## 🚀 Fitur

✅ CRUD Barang Elektronik

✅ Validasi data unik (nama & kode)  
✅ Pagination  
✅ Desain responsif (Tailwind + FluxUI)
